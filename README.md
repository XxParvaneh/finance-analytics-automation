# Finance Analytics and Process Automation Platform

An end-to-end portfolio project for financial data engineering,
analytics and process automation.

## Business Problem

A fictional company wants to consolidate financial transaction
and budget data, identify budget deviations, forecast future spending
and automate approval workflows.

## Planned Technology Stack

- Python
- Pandas
- SQL
- Snowflake
- dbt
- Power BI
- Power Apps
- Power Automate
- Dataverse
- GitHub Actions

## Project Status

Project foundation in progress.

## Disclaimer

All data used in this project is synthetic. The project contains no
confidential or real company data.
